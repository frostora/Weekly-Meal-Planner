About

This is a project to help you plan weekly meals, track leftovers, and generate shopping lists efficiently.

Create an account on the user page so as to save your meals across devices. Otherwise, you can use this service anonymously, but please be aware that your data will only be saved on this browser.

Note that this web app works best on desktop.

How it works

  1. Set up your ingredients in the Edit Ingredients page, assigning a portions count: how many meals that specific ingredient can be used for.
  2. Create your meals in the Edit Meals page, using those ingredients.
  3. Plan your week on the main page by selecting meals for each slot or leaving some empty for when you eat-out
  4. As the planner is filled,
    • A shopping list is dynamically generated
    • Leftovers are tracked
    • Suggestions help you use leftovers efficiently.
  5. Use, at the end of the week, the reset button at the bottom of the home page to start planning for the following one.
